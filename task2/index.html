Fix the weakest funnel step

Funnel: landing → sign up → first tool use → paywall → purchase.

Current state

There are two competing entry paths from the landing page:

  • Header "Get started" → sign up → empty dashboard → upload again → editor → download → paywall
  • Hero "Choose file" → editor immediately → download → auth → paywall

The hero path is product-led (value before account). The header path is account-led and adds friction: signup before any file interaction, then an empty "My documents" screen that asks users to upload a second time.

Landing → Sign up → First tool use → Paywall → Purchase

Weakest step

Paywall → purchase is the primary bottleneck. Both paths converge here at peak intent. The user has edited a document and sees "Your document is ready", yet pricing creates the largest drop.

  • Three plan tiers with unclear differences ("limited" vs "full" access)
  • Hidden renewal: $0.95/$1.95 trial, then $49.88/mo in fine print
  • Generic "Continue" CTA placed above plan selection
  • No pricing page on site; surprise pricing only at download moment

Secondary leak: sign up → first tool use on the header path (empty dashboard, second upload). Fixing entry increases qualified traffic to the paywall.

1

Paywall pricing redesign

Production paywall breaks trust: unclear tiers, hidden renewal, generic features. Users need clarity, not three plans to compare.

Variant A offers two options: trial for first-timers, annual for committed users. Renewal sits on the plan card; copy ties to the user's file. This captures annual upfront and tests whether simpler choice lifts conversion without dropping the plan step.

Variant B: payment step (no plan selection)

At download, the job is done: the file is ready and the user wants to leave with it. Most visitors came to finish one task, not to compare tiers. Plan selection at this moment adds friction when intent is highest.

With one default plan, there is nothing to choose. We skip the plan screen and send users straight to payment, removing one step from the path to download.

We accept that some upfront annual revenue is lost at checkout. Annual can be upsold later via trial-ending email, the cancel flow, and billing settings, once users understand the product. Variant A keeps choice for price-sensitive visitors; Variant B bets on higher checkout completion and recovers annual downstream.
